# MachiKania-AWPEN
MachiKania type P/PU のGROVE(Analog)に接続して使用するライトペン。
TVなど自己発光して画面表示するデバイスであれば使用できる。
タクトスイッチを押すタイミングで、照度センサーから光の検知を開始し、画面を4分割して、光っていたら更にその領域を４分割することを繰り返しタッチした場所を特定する。
